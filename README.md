# cursor-portfolio-repo

このリポジトリは、GitHub Pages でポートフォリオを公開するための置き場です。

## 公開するフォルダ

- `docs/` の中身が、そのままポートフォリオサイトです（`docs/index.html` が入口）

## GitHub Pages（無料）で公開する手順

1. GitHubで新規リポジトリを作成（例: `kyohei-mukaiyama2026`）
2. このフォルダの中身（`cursor-portfolio-repo`）を GitHub にアップロード
   - `docs/` フォルダごとアップロードしてください
3. GitHub リポジトリの `Settings` → `Pages`
4. `Build and deployment`
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/docs`
5. 保存して数分待つと、公開URLが表示されます

公開URLは通常こうなります:

- `https://<GitHubユーザー名>.github.io/<リポジトリ名>/`
